# Construccion-de-Software-y-toma-de-decisiones

A01711027
Repositorio individual del curso Construcción de Software y Toma de Decisiones del Tecnológico de Monterrey, Campus Querétaro. Aquí se almacenan las prácticas, laboratorios, ejercicios, evidencias y avances del proyecto integrador a lo largo del semestre.

CarreraÑ Ingeniería en Tecnologías ComputacionalesInstitución
Instituto Tecnológico y de Estudios Superiores de Monterrey

La unidad de formación (UF) Construcción de Software y Toma de Decisiones introduce a los principios fundamentales del desarrollo de aplicaciones web modernas, tanto del lado del cliente como del servidor, así como al uso de bases de datos para la gestión y análisis de información.
El enfoque de la materia es práctico: cada concepto teórico se acompaña de un laboratorio o ejercicio para aplicar lo aprendido construyendo software real, utilizando las herramientas y flujos de trabajo que se emplean en la industria. Y poder así, solucionar la problemática del socio formador o de la Organización Socio Formadora (OSF).

El flujo básico que se utiliza es:
git pull           # traer cambios del repositorio remoto
git add .          # preparar archivos modificados
git commit -m ""   # registrar un punto de control con descripción y el mensaje
git push           # subir cambios al repositorio remoto

Este repositorio es de carácter académico y forma parte del trabajo individual del semestre.
